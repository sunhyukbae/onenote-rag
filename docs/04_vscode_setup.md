# 3단계: VS Code + Claude Code 설정

## Claude Code for VS Code Extension 로그인

1. VS Code 실행
2. 왼쪽 사이드바 Claude 아이콘 클릭
3. `Sign in to Claude.ai` 클릭
4. Claude Pro 계정으로 로그인

---

## Claude Code 사용법

이 튜토리얼에서는 두 가지 방식으로 Claude를 사용한다:

| 방식 | 용도 |
|------|------|
| **Claude.ai 웹 채팅** | 설계 결정, 오류 원인 분석, 방향 설정 |
| **VS Code 내 Claude Code** | 실제 파일 작성, 코드 수정, 디버깅 |

---

## VS Code 터미널 여는 법

```
단축키: Ctrl + ` (백틱)
또는 상단 메뉴 → Terminal → New Terminal
```
